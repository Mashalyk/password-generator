# Password Generator

My first Python project.

This is a simple random password generator.

## How to run

1. Install Python
2. Download the project
3. Run:

python main.py

## What the program does

The program asks for the password length and generates a random password.
## Example

Enter password length: 12
How many passwords to generate: 2
Generated password: Xk9!dL2#pQz1
Generated password: aB7@Lm3!tQw9
